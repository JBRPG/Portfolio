Problem

Data management is a field and skill that is intensely associated with computer science. However, not too many data management tools are designed with beginners in mind.

I had the opportunity to work with the Local Ground team for Cornell University, to support the following task: How to manage data at browser level?

To best tackle the problem, the team and I had to instantiate the following principles:

  • Easily visualize data

  • Accessible for complete beginners

  • Simple to add, edit, and delete data

Goal

Given the problem space, I implemented a number of important features and improvements including, but not limited to:

  • Reducing steps for data management

  • Clean Aesthetic

  • Accessible

  • Easy to manage data

  • All done on browser
